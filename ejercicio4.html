Ejercicio 4: Lista de definición con estructura técnica.

Crea una lista de definición dl con al menos 5 conceptos clave de HTML.

  • Cada término dt debe contener una descripción (dd).
  • Dentro de cada descripción, incluye:
    • Una lista ordenada o no ordenada (dependiendo del contenido).
    • Enlaces externos con target="_blank".
    • Un ejemplode código entre pre code que demuestre cómo se usa la etiqueta explicada.

Resolución:

Elemento

Un elemento HTML es un componente individual de un documento HTML o XHTML. Está definido por una etiqueta de inicio, puede contener algún contenido y normalmente tiene una etiqueta de cierre.

  • Ejemplo: <p>Este es un párrafo</p>
  • Puedes aprender más sobre elementos HTML en MDN Web Docs.
Atributo

Los atributos HTML proporcionan información adicional sobre los elementos HTML. Se especifican en la etiqueta de inicio y suelen tener un nombre y un valor.

  1. El atributo class se utiliza para especificar una o más clases CSS para un elemento.
  2. El atributo id define un identificador único para un elemento.
  3. Lista completa de atributos en la documentación de MDN sobre atributos.

Ejemplo:

<a href="https://www.ejemplo.com" class="enlace-principal">Ejemplo</a>
Etiqueta

Las etiquetas HTML son palabras clave (o letras) encerradas entre corchetes angulares < y >. Indican al navegador cómo debe mostrar el contenido.

  • Las etiquetas suelen venir en pares: una etiqueta de inicio <p> y una etiqueta de cierre </p>.
  • Algunas etiquetas son auto-cerradas o vacías (<br>, <img src="...">).
  • Más información: W3Schools.

Ejemplo:

<img src="imagen.jpg" alt="Descripción de la imagen">
